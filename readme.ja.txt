== 説明 ==

このプラグインはカテゴリに属するすべての投稿を表示します。
階層構造を持つカテゴリの場合、すべての投稿がツリー状に表示されます（下記URLからサンプルを確認できます）。

http://charcoalphp.org/documents/

== 使用方法 ==

1. 管理ページにログインします。
2. 固定ページを作成します（タイトルは自由です）。
3. 下記のショートコードを本文に記載します:

[sspbc_posts_tree cat_id=10]

4. ページを公開します。
5. IDが10のカテゴリに保存された投稿が先ほど保存した固定ページにリスト表示されます。
